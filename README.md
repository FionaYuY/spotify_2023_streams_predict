# Project Title: Spotify 2023 Stream Predictions

## Project Description
This project analyzes the 'Most Streamed Spotify Songs of 2023' to identify patterns in streaming data and predict future streaming counts. The analysis includes exploratory data analysis, feature selection, and building a predictive model using a Random Forest regressor.
